# Minecraft Smooth Font

The Minecraft resource pack ``Smooth Font'' , which replace with a smooth font.

## Latest Version

1.0.0

## Smooth Fonts

### モリサワ BIZ UD ゴシック (Morisawa BIZ UD Gothic)

https://github.com/googlefonts/morisawa-biz-ud-gothic/

![モリサワ BIZ UD ゴシック](./images/Minecraft-BIZUDPGothic.png "モリサワ BIZ UD ゴシック")

### M+ Fonts

https://mplusfonts.github.io/

![M+ FONTS](./images/Minecraft-MPLUS.png "M+ FONTS")

## For Java Edition (1.21.9/1.21.10/1.21.11 or later) users

```
% git clone https://github.com/cvsync/SmoothFont

% cd SmoothFont

% /bin/sh build.sh

% ls 1.0.0/
SmoothFont-bizud.zip   SmoothFont-mplus.zip
```
